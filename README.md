# RAG Production App

A production-ready RAG (Retrieval-Augmented Generation) Agent application.

## Setup

1. Create a virtual environment:
```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

2. Install dependencies:
```bash
pip install -e .
```

## Development

Install development dependencies:
```bash
pip install -e ".[dev]"
```
